# YouTube REST API

A straightforward and effortless Restful API powered by ExpressJS, NodeJS, and, MongoDB.

<br />

## How to Run

In development mode :
```
yarn devStart
```

<br />

## Format code

You can format code using prettier :
```
yarn format
```
